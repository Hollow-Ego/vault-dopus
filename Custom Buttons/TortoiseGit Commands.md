
## Show Log

```
TortoiseGitProc.exe /command:log /path:{sourcepath}
```

## Commit

```
TortoiseGitProc.exe /command:commit /path:{sourcepath}
```

## Pull

```
TortoiseGitProc.exe /command:pull /path:{sourcepath}
```

## Switch/Checkout

```
TortoiseGitProc.exe /command:switch /path:{sourcepath}
```
